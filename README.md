# HR Attrition Analysis Dashboard

This project analyzes employee attrition using Power BI and HR dataset.

## Key KPIs
- Total Employees
- Employees Left
- Attrition Rate
- Active Employees
- Average Salary (Left)
- Average Age (Left)

## Visuals
- Attrition by Gender
- Attrition by Department
- Attrition by Job Role
- Attrition by Age Group
- Attrition by Tenure
- Employee Satisfaction Level

## Tools Used
- Power BI Desktop
- DAX
- Data Modeling
